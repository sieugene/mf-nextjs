### 1. Install project dependencies

Run in root folder:

`$ yarn install `

### 2. Start local development mode

Being in the project directory, execute in the console:

`$ yarn start `

### 3.  Open
`http://localhost:3000 - main app`

`http://localhost:3001 - cube app`

`http://localhost:3002 - prisma app`
